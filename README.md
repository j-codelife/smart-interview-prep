# Smart Interview Prep API

A backend REST API that simulates technical interview sessions.  
Users can start interview sessions, submit answers, receive feedback, and retrieve previous interview sessions.

This project demonstrates backend development concepts such as REST API design, database persistence, and modular server architecture.

---

## Tech Stack

- Node.js
- Express.js
- SQLite (better-sqlite3)
- REST APIs
- Git

---

## Features

- Start a new interview session
- Submit answers to interview questions
- Store interview sessions and responses in a relational database
- Retrieve previous interview sessions
- Modular backend architecture using Express routes

---

This project is currently **in progress**.
